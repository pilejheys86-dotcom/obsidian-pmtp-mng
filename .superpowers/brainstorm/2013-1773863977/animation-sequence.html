2. Animation Sequence (~12s loop)

A fake cursor moves through the mockup performing these actions in order, then loops:

1
Cursor enters → hovers "Total Active Loans" card
Card gets a subtle lift + border glow. Cursor rests for ~1.5s.
2
Value crossfade: $458,920 → $461,420
Loan amount fades to updated value. Trend indicator pulses green briefly.
3
Cursor moves → hovers "Inventory Value" card
Green card gets a brighter glow/scale effect. Cursor rests ~1s.
4
Cursor moves → hovers "Diamond Ring" row
Row highlights with a subtle background shift. Cursor rests ~1s.
5
Cursor "clicks" → brief ripple + "Approved" badge appears
A small green "Approved" pill fades in next to the price, cursor does a click animation (scale down then up).
6
Cursor moves → clicks "View All"
"View All" text pulses with primary color. Short pause, then everything resets and loops.

3. Implementation Summary

Theme Changes
  • Flip default: light mode is default, dark: prefix for dark
  • Light mode: white/gray surfaces, dark text
  • Dark mode: keeps current dark palette
  • Primary (green) card stays the same in both modes
Animation Changes
  • New CSS keyframes in index.css
  • Fake cursor div with position:absolute
  • Coordinated delays for hover/click/value-swap effects
  • Respects prefers-reduced-motion
  • ~12s total loop duration
